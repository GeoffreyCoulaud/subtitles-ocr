This directory contains example frames, where various subtitles are shown.

| File                                | Description                                                                           |
| ----------------------------------- | ------------------------------------------------------------------------------------- |
| 01-intro-logo                       | Introduction to an anime episode, with the anime logo and some fansub credits         |
| 02-opening-credits-and-karaoke      | During the opening, with translated + karaoke lyrics, and some fansub credits         |
| 03-intro-sponsors-1                 | After the opening, sponsor text                                                       |
| 04-intro-sponsors-2                 | After the opening, sponsor text                                                       |
| 05-episode-title-and-dialogue       | Some spoken dialogue, plus the episode's title translated in place (forced)           |
| 06-dialogue-and-forced-info         | Some spoken dialogue, plus a character's name translated in place (forced)            |
| 07-dialogue                         | Some spoken dialogue                                                                  |
| 08-colored-dialogue                 | Some spoken dialogue, colored blue (here to signal inner monologue)                   |
| 09-dialogue-and-translated-sign     | Some spoken dialogue, plus an in-place translated sign (under the drawn text)         |
| 10-colored-dialogue-and-forced-info | Some colored dialogue, plus a character's name translated in place (forced)           |
| 11-dialogue-and-translator-note     | Some spoken dialogue (at the bottom), plus a translator's note in purple (at the top) |
| 12-translated-sign                  | An in-place translated sign (next to the drawn text)                                  |

## Notes

Subtitles are in french in those frames.
The original material is an anime, dubbed in japanese, with japanese text in frame sometimes.
